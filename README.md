### Code Agent [![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) [![Flask](https://img.shields.io/badge/Flask-3.0-green)](https://flask.palletsprojects.com/)\n\n<!--- ![Demo](screenshots/demo.gif) -->\n\nA Flask web application that uses LLMs (Gemini, Ollama, OpenAI GPT, Claude) to **fix bugs, improve, and generate code**. Upload code/file + instruction → Get fixed code instantly. Supports file write-back.\n\n## 🚀 Features\n- 🔧 **Multi-LLM Support**: Gemini 1.5 Flash, Ollama (local), GPT-4o-mini, Claude 3.5 Sonnet\n- 📝 **Code Read/Write**: Paste code or file path, auto-saves fixes\n- 🖥️ **Web UI + API**: Simple HTML interface + POST /chat endpoint\n- ⚡ **Easy Setup**: pip install &amp;&amp; python app.py\n\n## 📋 Quick Setup\n1. Clone &amp; install: `pip install -r requirements.txt`\n2. Copy `.env.example` → `.env`, add your API keys\n3. Ollama: Install &amp; `ollama pull llama3` (optional local LLM)\n4. Run: `python app.py` → http://127.0.0.1:5000\n\n## 🔑 Environment Variables (.env)\n```\nGEMINI_API_KEY=your_gemini_key\nOPENAI_API_KEY=sk-...\nANTHROPIC_API_KEY=sk-ant-...\nOLLAMA_MODEL=llama3\n```\n\n## 💬 API Usage\n**POST** `/chat`\n\n```json\n{\n  "code": "print(&#39;buggy&#39;)",  // or file: &#39;app.py&#39;\n  "instruction": "Fix bug &amp; add logging.",\n  "llm": "gemini"  // gemini|ollama|openai|claude\n}\n```\n\nResponse: `{"response": "...", "fixed_code": "...", "write_status": "Saved to file"}`\n\n**cURL Example**\n```bash\ncurl -X POST http://127.0.0.1:5000/chat \\\n  -H "Content-Type: application/json" \\\n  -d &#39;{"code": "def add(a,b): return a+b", "instruction": "Handle str too", "llm": "ollama"}&#39;\n```\n\n## 🛠️ LLMs\n| LLM | Setup | Notes |\n| --- | --- | --- |\n| **Gemini** | `GEMINI_API_KEY` | Fast, free tier |\n| **Ollama** | Local install | Privacy, offline |\n| **OpenAI** | `OPENAI_API_KEY` | GPT-4o-mini |\n| **Claude** | `ANTHROPIC_API_KEY` | Excellent reasoning |\n\n## 🤝 Contributing\n1.
